# ControlSlider


## Description
The ControlSlider is a custom UIControl for iOS written in Swift 2.0.

## Requirements
- iOS 8.0+
- Xcode 7

## Installation


```ruby
use_frameworks!
pod 'ControlSlider', :git =>  'https://github.com/msedd/ControlSlider.git'
```
